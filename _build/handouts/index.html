Sales Report

Introduction

The file sales-report.txt contains details about recent melon orders. Each order (and hence, each line in sales-report.txt) contains the salesperson’s name, total amount of the order and total number of melons sold.

You’ll also find the sales_report.py script, which displays a report of the total number of melons sold by each salesperson.

Take a look at the sales_report.py script and figure out how it works. You don’t need to code a better solution right now, but imagine that you’re having a meeting with your manager very soon. Be prepared to talk about any inefficiencies you might find in the script (as in, what would be nice to have to make it better/faster?). In order to make the meeting go more quickly, comment each of the main tasks of the script to show that you understand what’s happening.
